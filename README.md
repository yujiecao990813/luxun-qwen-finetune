🎭 基于 Qwen2-0.5B 的“鲁迅”人格大模型微调 (Luxun-Qwen-Finetune)
本项目通过 ms-swift 框架对 Qwen2-0.5B-Instruct 进行有监督微调 (SFT)，旨在通过深度学习技术，让大模型习得并模拟鲁迅先生的行文风格、思想深度与标志性反讽笔触。

📖 项目简介
本实验不仅是一个简单的模型微调示例，更是一次文学与技术的碰撞。通过注入精心构思的语料库，模型成功捕捉了鲁迅先生在现代叙事语境下的特有逻辑与审美，在对话中展现出冷峻的反思与人文关怀。

📊 数据集 (Dataset)
数据存放于 datasets/luxun_dataset.json。

涵盖主题：内卷、现代消费心理、精神内耗、人工智能文学等。

语料质量：精选并清洗的问答对，重点训练模型对“反讽”与“深刻批判”的把握能力。

🚀 快速上手 (Quick Start)
环境依赖
确保已安装最新版 ms-swift：

Bash
pip install ms-swift[llm]
1. 模型训练 (SFT)
直接执行仓库内的脚本进行微调，该脚本已配置最优化的学习率与批量大小：

Bash
bash train_sft.sh
2. 模型交互 (Infer)
训练完成后，使用生成的 Checkpoint 进行交互推理：

Bash
bash infer_chat.sh
🧠 核心技术栈
基座模型: Qwen2-0.5B-Instruct

训练框架: ms-swift (阿里巴巴开源的高效 LLM 训练推理框架)

技术策略: Supervised Fine-Tuning (SFT)

性能优化: Cosine 学习率调度、低资源环境适配

🧪 效果展示 (Sample)
用户: 先生，现在的年轻人都在追求‘情绪价值’，您怎么看？
模型: “……盲目追求所谓的‘情绪价值’，情感与理智并重，方能做出更为明智的选择。先生之言，意在提醒我们，无论何时何地，都应保持清醒的头脑，拒绝盲从或随波逐流。”
